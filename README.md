# Angular Studies

An Angular Studies repo.


## Dependencies
* [Node.js and npm](https://nodejs.org/en/download/)
* [Angular-cli](https://cli.angular.io/)

## How to run
### application
```bash
# on alurapic directory run:
npm install
```

### api

```bash
# on api directory run:
npm install
```

## How to run
### application
```bash
# on alurapic directory run:
npm start
```

### api

```bash
# on api directory run:
npm start
```
